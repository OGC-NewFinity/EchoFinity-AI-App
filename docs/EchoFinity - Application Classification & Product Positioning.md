# EchoFinity - Application Classification & Product Positioning

**Developer**: OGC NewFinity  
**Application Name**: EchoFinity  
**Category**: AI-Powered Video Creation & Editing Platform  
**Target Markets**: Content Creators, Storytellers, Marketing Professionals, Social Media Creators

---

## Executive Overview

EchoFinity is a comprehensive AI-powered video creation and editing platform designed for anyone who creates visual content. Unlike specialized tools that focus on a single use case, EchoFinity serves the entire content creation lifecycle: from raw footage capture to professional post-production editing, all powered by artificial intelligence.

The platform enables users to record video directly within the app, apply AI-driven editing and enhancement, and export professional-quality content in 30-second to 2-minute formats optimized for social media and marketing campaigns. The application combines the speed of AI automation with the creative control of manual editing, making professional video production accessible to creators of all skill levels.

### Token System

Tokens represent computational resources consumed during video processing. Different operations consume different token amounts:

| Operation | Token Cost | Notes |
|---|---|---|
| Video Recording (per minute) | 20 tokens | Includes storage and processing |
| AI Editing (per video) | 30-50 tokens | Varies by length and complexity |
| Color Correction | 10 tokens | Per application |
| Subtitle Generation | 15 tokens | Per video |
| Export (720p) | 5 tokens | Per export |
| Export (1080p) | 10 tokens | Per export |
| Export (4K) | 20 tokens | Per export |
| Cloud Storage (per GB/month) | 1 token | Automatic deduction |

**Token Refresh**: Tokens refresh daily at midnight (user's local timezone). Unused tokens do not roll over to the next day.

## Technical Architecture Overview

### System Components

**Mobile Frontend** (Flutter - Dart)
- iOS and Android native apps
- Real-time video preview and recording
- Timeline-based editing interface
- Offline-capable with sync

**Backend API** (Node.js + Express)
- User authentication and authorization
- Project management and storage
- Token system and billing
- API endpoints for mobile and web

**AI Processing Service** (Python + FastAPI)
- Video analysis and scene detection
- Color grading and correction
- Subtitle generation
- Music and sound effect recommendations
- Pacing optimization

**Media Processing** (FFmpeg + Node.js)
- Video encoding and transcoding
- Format conversion and optimization
- Watermark application
- Batch processing

**Cloud Infrastructure** (AWS)
- S3 for video and asset storage
- Lambda for serverless processing
- RDS PostgreSQL for data persistence
- CloudFront for content delivery
- SQS for job queuing
